# The Rexy Programming Language

*A Tutorial*

Rexy is a small, modern systems language. This book is for programmers
who already know at least one language and want to learn how to read
and write Rexy.

It teaches Rexy by example. Every chapter gives you code to type, a
way to run it, and the output you should see. By the end of the book
you will have written, run, and modified Rexy programs that touch
every feature the language currently provides.

You will see Rexy as a systems language. Memory is visible. Mutation
is opt-in. Pointer access lives behind an `unsafe` boundary. Types are
written down. The book covers those choices on purpose, because they
are the choices that make Rexy a useful systems language.

**Drew Tennenbaum**
