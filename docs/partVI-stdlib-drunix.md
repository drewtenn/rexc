# Part VI - The Standard Library and Drunix

The previous five parts cover the language itself. This final part turns
the language outward. Chapter 13 tours the standard library: the helpers
Rexy makes available without ceremony for console I/O, string handling,
and parsing. Chapter 14 closes the book by walking a Rexy program through
the Drunix toolchain, from `.rx` source to a runnable ELF executable.

By the end of Part VI, the reader has met every piece of Rexy that this
book is going to introduce. They can write a program, compile it, link it
into a Drunix-compatible executable, and reason about how it interacts
with the operating system that hosts it.
