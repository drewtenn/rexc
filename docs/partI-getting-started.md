# Part I - Getting Started With Rexy

Before you can do anything interesting in Rexy, you need a program that
runs. Part I gets you from "I have never written a Rexy file" to "I can
write small Rexy programs that compute and return a value".

Chapter 1 walks you through the smallest possible Rexy program: you
write it, build it, run it, and see what it returns. Chapter 2
shows you the primitive values you can put into a program: integers,
booleans, characters, and strings, with examples for each. Chapter 3
introduces bindings and functions, the two constructs that turn isolated
values into programs that compute.

By the end of Part I, you will have a Rexy file with several functions,
local variables, mutation where you need it, and a `main` that computes
and returns a result. That is the floor everything else in the book
builds on.
