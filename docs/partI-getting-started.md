# Part I - Getting Started With Rexy

Before we can talk about what makes Rexy interesting, we need a program that
runs. Part I gets the reader from "I have never written a Rexy file" to "I can
write small Rexy programs that compute and return a value".

Chapter 1 walks through the smallest possible Rexy program, builds it, and
explains what each piece is doing. Chapter 2 introduces Rexy's primitive
values: integers, booleans, characters, strings, and how to convert between
them. Chapter 3 covers bindings, functions, and the rules Rexy enforces about
mutation and types.

By the end of Part I, the reader can write a Rexy file with multiple
functions, declare local variables, mutate them where needed, and return a
result. That is the floor we will build everything else on.
