# Part VI - The Standard Library and the Toolchain

The previous five parts covered the language itself. This final part
turns the language outward. Chapter 13 tours the standard library: the
helpers Rexy makes available without ceremony for console I/O, string
handling, and parsing. Chapter 14 closes the book by walking a Rexy
program through the full toolchain, from `.rx` source to a runnable
executable on disk.

By the end of Part VI, you will have met every piece of Rexy that
this book is going to introduce. You can write a program, compile it,
link it into a runnable executable, and reason about how it interacts
with the operating system that hosts it.
